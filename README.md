# NEE Travel Intelligence Dashboard 🌍✈️

### Premium Market Intelligence for Travel Services (LATAM → Europe)

A high-end, responsive, and data-driven dashboard designed for travel agencies focusing on the Latin America to Europe corridor. This application provides real-time insights using RapidAPI (TikTok Scraper) and search intelligence (Google Trends).

## 🚀 Key Features

-   **Global Market Overview**: Live tracking of travel interest growth across different industries.
-   **Travel Intelligence Deep-Dive**: Analysis of travel subsegments (Adventure, Gastro, Luxury) across social platforms.
-   **Social Profile Analytics (The Gates)**: Real-time authentication via RapidAPI to fetch engagement, followers, and niche performance.
-   **Destination Rankings**: Monthly updated rankings of top European destinations for LATAM travelers.
-   **Strategic Insights**: AI-derived actionable business recommendations.
-   **Mobile-First Design**: Premium glassmorphism UI optimized for on-the-go monitoring.

## 🛠️ Technology Stack

-   **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
-   **Charts**: Chart.js for interactive visualizations.
-   **Maps**: Leaflet.js with custom tile layers.
-   **APIs**: RapidAPI (TikTok Scraper), Google Trends Intelligence.
-   **Design**: Modern Dark Mode, Backdrop blurs, CSS Grid/Flexbox.

## 📦 Deployment

1.  Push code to GitHub.
2.  Enable **GitHub Pages** in Settings > Pages.
3.  Set source to `main` branch.
